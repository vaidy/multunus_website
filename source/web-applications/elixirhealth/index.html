ElixirHealth.com: This web application focuses on the “wellness”, “consumer” and “e-commerce” aspects of the Health vertical. Therefore its features include easily browseable and searchable Health Content, a comprehensive directory of Health Care providers across the US, a marketplace which allows Consumers and Health Care providers to interact and trade online – among other features.

Technologies Used: Ruby on Rails [Rails 3 and RSpec 2], Rails Engines, Sunspot [Ruby API for working with Solr], JSTestDriver and Jasmine.
